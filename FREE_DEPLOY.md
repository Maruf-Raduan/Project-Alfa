# 100% FREE Deployment Options

## Option 1: Render.com (EASIEST - Recommended)
**Free tier: 750 hours/month, auto-sleep after 15 min**

### Steps:
1. Push code to GitHub
2. Go to https://render.com
3. Click "New +" → "Web Service"
4. Connect your GitHub repo
5. Click "Deploy"
6. Done! Get your URL: `https://java-assistant.onrender.com`

**No credit card required!**

---

## Option 2: Railway.app
**Free tier: $5 credit/month**

### Steps:
1. Go to https://railway.app
2. Click "Start a New Project"
3. Select "Deploy from GitHub repo"
4. Connect repo and deploy
5. Get URL from dashboard

---

## Option 3: Vercel (For static + serverless)
**Free tier: Unlimited**

### Steps:
```bash
npm i -g vercel
vercel login
vercel
```

---

## Option 4: PythonAnywhere
**Free tier: 1 web app**

### Steps:
1. Sign up at https://www.pythonanywhere.com
2. Upload files via "Files" tab
3. Go to "Web" tab → "Add a new web app"
4. Choose Flask, set path to app.py
5. Reload and visit your URL

---

## Option 5: Replit (INSTANT - No setup)
**Free tier: Public repls**

### Steps:
1. Go to https://replit.com
2. Click "Create Repl" → Import from GitHub
3. Paste your repo URL
4. Click "Run"
5. Share the URL!
